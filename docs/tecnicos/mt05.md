# MT05 Impresión 3d
En este módulo exploramos la impresión 3d y el escaneo laser.
Uno de los principales beneficios de esta disciplina es que es una fabricación aditiva que genera desperdicio casi nulo. A continuación, comparativo contra la fabricación sustractiva: 
![](../images/MT05/01.%20comparativo.jpg)

La tarea de este módulo es preparar para impresión 3d, el volumen de un engranaje modelado en Fusion 360, haciendo los siguientes pasos. Debemos evaluar la manera más efectiva de imprimir el engranaje cambiando parámetros de impresión.
1.	Convertir el archivo nativo fusión 360 en *stl (binario) grabándolo como malla.
![](../images/MT05/02.%20fusion.jpg)
2. Abrir el archivo en el programa de laminación de volúmenes 3d IdeaMaker. Para el ejercicio trabajamos con IdeaMaker aunque en el laboratorio de ESAN Perú trabajamos con el programa Bambú Studio.
3.	Escalamos uniformemente el objeto a 150%
![](../images/MT05/03.%20escala.jpg)
4.	Aseguramos que este seleccionado el extrusor izquierdo con la impresora Raise Pro2 Plus
![](../images/MT05/04.%20extrusor%20izq.jpg)
5.	Afinamos los siguientes parámetros:
![](../images/MT05/05.%20parametros.jpg)
6.	Lanzamos el laminado y podemos visualizar el tiempo de impresión, la cantidad de material utilizado y el costo.
7.	Con los mismos parámetros se hicieron 4 laminados cambiando la orientación del engranaje contra la superficie de impresión.
Esto muestra como el cambio de orientación afecta el tiempo de impresión, cantidad de material y costo.

#### Conclusión
Aunque la posición 3 sea más eficiente optaría por la posición 1 por tener las capas mejor adaptadas a la geometría del objeto. Al ser un engranaje debe tener mayor rigidez.
