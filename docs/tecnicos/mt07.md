# MT07 Interfases y Aplicaciones

Este módulo es la Introducción a los principios de programación y electrónica aplicada a Interfases y Aplicaciones. Interfaz es un puente entre el sistema físico y quien lo usa, que permite ver estados y ejecutar acciones.

![](../images/MT07/01.%20CAPAS%20DE%20INTERFAZ.jpg)

#### INTERFAZ GRÁFICA DE USUARIO (GUI)
Una interfaz gráfica de usuario es el medio visual que permite al usuario interactuar con un sistema o aplicación mediante elementos gráficos en lugar de comandos de texto.
El **frontend** es la parte visible de una aplicación o sistema, lo que el usuario ve y con lo que interactúa directamente.
El **backend** es la parte interna del sistema, donde se procesan los datos y se ejecuta la lógica. No es visible para el usuario.

#### ALTERNATIVAS PARA CREAR APLICACIONES
Revisamos diversas herramientas digitales para crear aplicativos. Desde las más básicas como **MIT APP INVENTOR** que no requiere código, pasando por **Streamlit** que requiere saber de Python, **Lovable** que utiliza IA para generar APP’s a partir de promts, **Blynk** que permite ver dashboards en Iot y **Arduino full stack** que genera un ecosistema completo que conecta la placa con la nube y el aplicativo.

Todavía estoy iterando el funcionamiento del juego, pero básicamente se lanza el dado y se avanza las casillas, en cada casilla hay retos que te hacen avanzar o retroceder. El ganador es quien llega al meta primero.

El hardware mínimo será:
•	ESP32 DevKit V1 → microcontrolador con WiFi y Bluetooth integrado.
•	Protoboard pequeña + cables jumper → para conexiones rápidas sin soldar.
•	2 LEDs (rojo y verde) + resistencias de 220 Ω → para pruebas de encendido/apagado.
•	Batería portátil (power bank) o módulo recargable → para alimentar el ESP32 sin PC.
•	Móvil con app Lovable instalada
